# 🎮 Fall Guys SPedi - 3D OpenGL Game

A fun 3D racing and obstacle course game inspired by Fall Guys, built using Python and OpenGL.

## 📋 Game Features

- **3 Difficulty Levels**: Easy, Medium, and Hard
- **Hard Mode (Special)**: 3 layered arena with vanishing tiles (Hex-A-Gone style)
- Moving obstacles: Spinning hammers, sliding doors, pendulum hammers, floating hammers
- Launch pads that throw you forward
- AI opponents that race with you
- Shooting and Sword attack system
- Score system with time and rank bonuses

## 🎯 Controls

| Key / Action          | Function                    |
|-----------------------|-----------------------------|
| **WASD**              | Move forward/back/left/right |
| **Mouse Move**        | Rotate player               |
| **Left Click**        | Shoot bullets               |
| **Right Click**       | Toggle Sword                |
| **Space**             | Jump                        |
| **C**                 | Change Camera Mode          |
| **R**                 | Restart Game                |
